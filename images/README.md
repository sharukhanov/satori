## Как заменить фото

1. Откройте эту папку `images/` на GitHub.
2. Нажмите **Add file → Upload files**.
3. Перетащите фото. Чтобы оно сразу встало на нужное место — назовите его
   так же, как в таблице выше (например, `service-hair.jpg`).
4. Нажмите **Commit changes**.

> Совет: оптимизируйте фото перед загрузкой (ширина ~1600 px, формат JPG,
> «вес» до ~300 КБ) — сайт будет загружаться быстрее.
