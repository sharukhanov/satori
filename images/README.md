## Фирменная надпись «Сатори» (каллиграфия)

Имя «Сатори» — это **готовая надпись-каллиграфия, а не шрифт**. Используются файлы:

| Файл                          | Где                                    | Примечание                                              |
|-------------------------------|----------------------------------------|---------------------------------------------------------|
| `satori-wordmark.svg`         | Шапка и hero (на светлом фоне)          | Векторная надпись, цвет задаётся через `fill` в SVG     |
| `satori-wordmark-light.svg`   | Подвал (на тёмном изумрудном фоне)      | Светлый вариант надписи                                 |

Сейчас в обоих файлах — аккуратная заглушка антиквой. Замените их на свою
готовую векторную надпись (можно PNG с прозрачным фоном — тогда поменяйте
имя файла в `src` у соответствующего `<img>` в `index.html`).

## Видео-логотип в hero (по желанию)

Рядом с надписью «Сатори» на первом экране предусмотрен круг-энсо под
зацикленное видео-лого. Пока файлов нет — в круге показывается мягкая
подсказка. Чтобы включить видео, загрузите в `images/`:

| Файл                       | Назначение                                        |
|----------------------------|---------------------------------------------------|
| `satori-logo.webm`         | Видео-лого (основной формат)                       |
| `satori-logo.mp4`          | Видео-лого (запасной формат для Safari)            |
| `satori-logo-poster.jpg`   | Кадр-заглушка, пока видео грузится                 |

Видео должно быть **квадратным (1:1)** с кремовым фоном под цвет сайта.
Автозапуск уже настроен (`autoplay muted loop playsinline`).

## Как заменить фото

1. Откройте эту папку `images/` на GitHub.
2. Нажмите **Add file → Upload files**.
3. Перетащите фото. Чтобы оно сразу встало на нужное место — назовите его
   так же, как в таблице выше (например, `service-hair.jpg`).
4. Нажмите **Commit changes**.

> Совет: оптимизируйте фото перед загрузкой (ширина ~1600 px, формат JPG,
> «вес» до ~300 КБ) — сайт будет загружаться быстрее.
